# Инструкция по созданию иконок разных размеров

## Требуемые файлы:
- `logo-16.png` - 16x16px (для иконки в тулбаре)
- `logo-32.png` - 32x32px (для боковой панели)
- `logo-48.png` - 48x48px (для уведомлений)
- `logo-128.png` - 128x128px (для Chrome Web Store)

## Инструкция:

1. Откройте https://squoosh.app/
2. Загрузите исходный файл `logo.png`
3. Для каждого размера:
   - Выберите формат: PNG
   - Установите размер: 16x16, 32x32, 48x48, 128x128
   - Включите опцию "Resize" и установите точные размеры
   - Для качества выберите "Lossless" или качество 90-100
   - Сохраните как `logo-16.png`, `logo-32.png`, `logo-48.png`, `logo-128.png`
4. Поместите все файлы в папку `aegis_start/icons/`

## Альтернативный способ (ImageMagick):
```bash
magick logo.png -resize 16x16 logo-16.png
magick logo.png -resize 32x32 logo-32.png
magick logo.png -resize 48x48 logo-48.png
magick logo.png -resize 128x128 logo-128.png
```

## Проверка:
После создания файлов перезагрузите расширение в Chrome.
