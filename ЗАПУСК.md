# 🚀 Запуск Aegis - Просто и быстро!

## 📁 Единые файлы запуска

Созданы 3 файла для автоматического запуска сервера:

### 🪟 Windows:
```bash
start_aegis.bat
```
**Просто дважды кликните на файл!**

### 🐧 Linux/macOS:
```bash
./start_aegis.sh
```

### 🐍 Python (универсальный):
```bash
python start_aegis.py
```

## 🎯 Что делают эти файлы:

1. **Проверяют зависимости** - Python 3.8+, папки проекта
2. **Устанавливают пакеты** - автоматически из requirements.txt
3. **Запускают сервер** - на http://127.0.0.1:8000
4. **Показывают ссылки** - документация, админ панель, статус

## 🌐 После запуска доступно:

- **API**: http://127.0.0.1:8000
- **Документация**: http://127.0.0.1:8000/docs  
- **Admin UI**: http://127.0.0.1:8000/admin/ui
- **Статус**: http://127.0.0.1:8000/health

## 📱 Установка расширения:

1. Откройте Chrome → Настройки → Расширения
2. Включите "Режим разработчика"
3. Нажмите "Загрузить распакованное расширение"
4. Выберите папку `aegis_start`

## ✅ Готово!

Теперь у вас есть единый файл для запуска всего проекта! 🎉

**Для остановки сервера нажмите Ctrl+C**
