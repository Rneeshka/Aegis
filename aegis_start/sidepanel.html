О проекте AEGIS

AEGIS — современное расширение для безопасного серфинга. Мы анализируем URL, ссылки по наведению и загрузки, чтобы вовремя предупредить о фишинге и вредоносных ресурсах.

  • URL и hover-анализ в реальном времени
  • Уведомления об угрозах
  • Поддержка аккаунтов и API ключей
  • Приватность и локальные эвристики
