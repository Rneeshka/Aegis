# Настройка переменных окружения

## Быстрая настройка

1. Скопируйте файл `env.local` в `.env`:
   ```bash
   cp env.local .env
   ```

2. Или создайте `.env` вручную и скопируйте содержимое из `env.local`

## Важные переменные

### Обязательные:
- `DATABASE_URL` - URL базы данных PostgreSQL
- `ADMIN_JWT_SECRET` - Секретный ключ для JWT токенов (ОБЯЗАТЕЛЬНО измените!)

### Рекомендуемые:
- `ADMIN_ALLOWED_ORIGINS` - Добавьте все домены, включая:
  - `https://avqonadmin.com`
  - `https://admin.avqon.com`
  - `https://avqonadmin.avqon.com`
  - И другие ваши домены

### Безопасность:
- `ADMIN_DEFAULT_PASSWORD` - Измените пароль по умолчанию в production
- `ADMIN_DANGER_ZONE_PASSWORD` - Пароль для опасной зоны (полная очистка БД)

## Проверка настроек

После создания `.env` файла запустите:
```bash
python first_setup.py
```

Этот скрипт проверит:
- Подключение к базе данных
- Наличие необходимых таблиц
- Покажет информацию о пользователе по умолчанию
