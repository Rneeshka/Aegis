# 🔄 ПЕРЕЗАПУСТИТЕ СЕРВЕР

## Что я исправил:
1. Упростил endpoint `/health` для быстрого ответа
2. Исправил импорт `config` вместо `external_config`

## Что нужно сделать:

### 1. Остановите старый сервер
Нажмите **Ctrl+C** в терминале где запущен сервер

### 2. Запустите сервер заново:
```bash
python start_aegis.py
```

### 3. Перезагрузите расширение
1. Откройте `chrome://extensions/`
2. Найдите "SafeBrowse"
3. Нажмите **🔄 Перезагрузить**

### 4. Откройте popup расширения
1. Кликните на иконку расширения
2. Должно появиться: "✅ Подключено к API"
3. Кнопки должны заработать

## Если все еще не работает:

### Проверьте Console:
1. Откройте DevTools (F12) в popup расширения
2. Правый клик на иконке → "Проверить всплывающее окно"
3. Посмотрите ошибки в Console

### Проверьте сервер:
Откройте в браузере: http://127.0.0.1:8000/health
- Должен вернуться JSON с "status": "success"

### Проверьте CORS:
В браузере откройте DevTools → Network
- Запрос к `/health` должен возвращать 200 OK
- Headers должны содержать `Access-Control-Allow-Origin: *`
