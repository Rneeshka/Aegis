# 🛡️ Aegis - Антивирусное браузерное расширение

Система защиты от вредоносных URL и файлов с поддержкой Free/Premium режимов

## 🚀 Быстрый старт

### 1. Запуск бэкенда
```bash
cd antivirus-core
.\run_local.ps1
```

### 2. Установка расширения
1. Откройте Chrome → Настройки → Расширения
2. Включите "Режим разработчика"
3. Нажмите "Загрузить распакованное расширение"
4. Выберите папку `aegis_start`

### 3. Доступ к интерфейсам
- **API**: http://127.0.0.1:8000
- **Документация API**: http://127.0.0.1:8000/docs
- **Admin UI**: http://127.0.0.1:8000/admin/ui
- **Статус системы**: http://127.0.0.1:8000/health

## 🔑 Система ключей

### 🆓 Free (Бесплатно)
- ✅ Базовая проверка URL
- ✅ Анализ файлов по хешу
- ✅ Проверка доменов
- ✅ Работает без API ключа

### 💎 Premium (Платно)
- ✅ Анализ по наведению (hover analysis)
- ✅ Расширенная аналитика через внешние API
- ✅ Проверка IP адресов
- ✅ Приоритетная обработка
- ✅ Требует API ключ

## 🎛️ Admin UI

Управление системой через веб-интерфейс:
- **Создание премиум-ключей** с настройкой срока действия
- **Продление существующих ключей**
- **Просмотр статистики** использования
- **Отображение оставшегося времени** действия ключей

## 📊 API Endpoints

### Основные
- `POST /check/url` - проверка URL (работает без ключа)
- `POST /check/file` - анализ файла по хешу (работает без ключа)
- `GET /check/domain/{domain}` - проверка домена (работает без ключа)

### Premium (требует ключ)
- `POST /check/hover` - анализ по наведению
- `GET /check/ip/{ip}` - проверка IP адреса

### Admin
- `GET /admin/ui` - веб-интерфейс админ панели
- `POST /admin/api-keys/create` - создание ключей
- `POST /admin/api-keys/extend` - продление ключей

## 🔧 Функциональность

### Браузерное расширение
- ✅ Сканирование ссылок при наведении (premium)
- ✅ Проверка файлов при загрузке
- ✅ Уведомления о угрозах
- ✅ Настройки Free/Premium режимов
- ✅ Автоматическая реинъекция после обновления

### Backend API
- ✅ Интеграция с VirusTotal, Google Safe Browsing, AbuseIPDB
- ✅ Система лимитов и аутентификации
- ✅ Кэширование результатов
- ✅ Логирование и мониторинг

## 🛠️ Технологии

- **Frontend**: JavaScript, Chrome Extensions API
- **Backend**: Python, FastAPI, SQLite
- **Внешние API**: VirusTotal, Google Safe Browsing, AbuseIPDB
- **База данных**: SQLite с WAL режимом
- **Кэширование**: In-memory + Disk cache
