# ✅ Регистрация и Вход в Расширении

## 🎯 Что реализовано

Добавлены формы регистрации и входа в настройки расширения (`options.html` и `options.js`).

### Новые возможности:
- ✅ **Форма входа** — авторизация в существующем аккаунте
- ✅ **Форма регистрации** — создание нового аккаунта с привязкой ключа
- ✅ **Информация об аккаунте** — отображение username и email
- ✅ **Автоматическое сохранение API ключа** после успешной регистрации/входа
- ✅ **Выход из аккаунта** — кнопка для выхода

## 🚀 Как использовать

### 1. Регистрация нового аккаунта

1. Откройте настройки расширения (правой кнопкой по иконке → Настройки)
2. Прокрутите до секции **"Аккаунт Premium"**
3. Нажмите **"Регистрация"**
4. Заполните все поля:
   - **Username** — уникальное имя пользователя
   - **Email** — ваша почта
   - **Пароль** — минимум 6 символов
   - **API ключ** — премиум ключ, который нужно привязать
5. Нажмите **"Зарегистрироваться"**

После успешной регистрации:
- ✅ Аккаунт создается в системе
- ✅ API ключ автоматически привязывается к аккаунту
- ✅ API ключ автоматически сохраняется в настройках
- ✅ Вы автоматически входите в аккаунт

### 2. Вход в аккаунт

1. Если вы уже зарегистрированы, нажмите **"Войти"**
2. Введите **Username** (или email) и **Пароль**
3. Нажмите **"Войти"**

После успешного входа:
- ✅ Отображается информация об аккаунте
- ✅ Привязывается API ключ (если есть привязанные ключи)
- ✅ API ключ сохраняется в настройках

### 3. Выход из аккаунта

Нажмите кнопку **"Выйти"** в секции информации об аккаунте.

## 🔧 Технические детали

### API Endpoints используемые в расширении:

1. **POST /auth/register** — Регистрация
   ```javascript
   {
     username: "user123",
     email: "user@example.com",
     password: "pass123",
     api_key: "PREMI-..."
   }
   ```

2. **POST /auth/login** — Авторизация
   ```javascript
   {
     username: "user123",
     password: "pass123"
   }
   ```

### Данные сохраняются в `chrome.storage.sync`:
- `account` — информация об аккаунте (username, email, id)
- `apiKey` — API ключ привязанный к аккаунту

### Переключение между формами:
- **"Регистрация"** → показывает форму регистрации
- **"Вернуться к входу"** → показывает форму входа
- Автоматическое переключение после успешной авторизации

## 💡 Важные моменты

1. **API ключ для регистрации** — нужен премиум ключ, созданный в Admin UI
2. **API Base URL** — должен быть указан в настройках перед регистрацией
3. **Автоматическая привязка** — ключ привязывается к аккаунту и становится недоступен другим
4. **Минимальный пароль** — 6 символов
5. **Уникальность** — username и email должны быть уникальными
