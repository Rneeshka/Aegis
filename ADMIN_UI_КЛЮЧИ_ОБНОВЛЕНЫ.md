# ✅ Admin UI - страница ключей обновлена

## Что добавлено:

### 🔑 **Расширенная информация о ключах**:

#### Новые колонки в таблице:
1. **Ключ** - полный API ключ (как было)
2. **Имя** - название ключа (как было)
3. **Активен** - статус активности (как было)
4. **Уровень** - премиум/базовый (как было)
5. **Username** - имя пользователя аккаунта
6. **Email** - email аккаунта
7. **Пароль** - показывает "***" если есть пароль, "-" если нет
8. **Лимиты (день/час)** - дневные и часовые лимиты (как было)
9. **Запросы (сегодня/час)** - количество запросов сегодня и за час (как было)
10. **Всего** - общее количество запросов (как было)
11. **Последнее использование** - время последнего использования (как было)
12. **Истекает** - дата истечения (как было)
13. **Осталось** - время до истечения (как было)

### 📊 **Обновленный SQL запрос**:
```sql
SELECT api_key, name, is_active, access_level, rate_limit_daily, rate_limit_hourly,
       requests_total, requests_today, requests_hour, created_at, last_used, expires_at,
       user_id, 
       (SELECT username FROM accounts WHERE accounts.id = api_keys.user_id) as username,
       (SELECT email FROM accounts WHERE accounts.id = api_keys.user_id) as email,
       (SELECT password_hash FROM accounts WHERE accounts.id = api_keys.user_id) as password_hash
FROM api_keys
ORDER BY created_at DESC
LIMIT 200
```

### 🔒 **Безопасность**:
- **Пароль не показывается** в открытом виде
- **Показывается "***"** если пароль есть
- **Показывается "-"** если пароля нет
- **Email показывается** для связи с пользователем

## Как использовать:

### 1. **Просмотр ключей**:
1. Откройте **Admin UI**: http://127.0.0.1:8000/admin/ui
2. Перейдите в раздел **"Ключи API"**
3. Увидите **расширенную таблицу** с информацией об аккаунтах

### 2. **Информация об аккаунтах**:
- **Username**: имя пользователя (или "Свободен" если не привязан)
- **Email**: email аккаунта (или "-" если нет аккаунта)
- **Пароль**: "***" если есть пароль, "-" если нет

### 3. **Управление ключами**:
- **Создание**: только премиум ключи
- **Продление**: через форму продления
- **Просмотр**: полная информация об использовании
