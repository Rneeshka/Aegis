## Как проверить работу:

### 1. Запустите сервер:
```bash
# Вариант 1: Через bat файл
run_server_fixed.bat

# Вариант 2: Через PowerShell
.\run_server.ps1

# Вариант 3: Напрямую
cd antivirus-core
python -m uvicorn app.main:app --host 127.0.0.1 --port 8000 --reload
```

### 2. Проверьте сервер:
```bash
python test_all_functions.py
```

### 3. Обновите расширение:
1. **Удалите старое расширение** из `chrome://extensions/`
2. **Загрузите заново** из папки `aegis_start`
3. **Перезагрузите расширение**

## Тестирование функций:

### Восстановление пароля:
1. Откройте настройки расширения
2. Нажмите "Забыли пароль?"
3. Введите email → получите код (в логах сервера)
4. Введите код и новый пароль → готово!

### Проверка истечения ключей:
1. Создайте ключ с коротким сроком в Admin UI
2. Дождитесь истечения
3. Попробуйте использовать ключ → должен быть заблокирован

### Admin UI:
- **URL**: http://127.0.0.1:8000/admin/ui
- **Функции**: создание ключей, продление, просмотр статистики

## Проверка расширения:

### Если расширение не подключается:
1. **Проверьте консоль браузера** (F12 → Console)
2. **Проверьте статус сервера**: http://127.0.0.1:8000/health
3. **Перезагрузите расширение** в `chrome://extensions/`
4. **Проверьте настройки** расширения

### Если настройки не открываются:
1. **Проверьте popup.js** - исправлен дублирующийся код
2. **Перезагрузите расширение**
3. **Проверьте консоль** на ошибки JavaScript
