## 📋 Принцип работы

### Базовая версия
- Ключ создается свободным (`user_id = NULL`)
- Работает без аккаунта
- Доступен базовый функционал

### Премиум версия
1. Администратор создает премиум-ключ
2. Пользователь регистрируется через `/auth/register` с указанием ключа
3. Ключ автоматически привязывается к аккаунту
4. Ключ больше не доступен другим аккаунтам
5. Пользователь входит через `/auth/login`

## 🔌 Быстрый старт

### Создать аккаунт и привязать ключ:
```bash
curl -X POST "http://127.0.0.1:8000/auth/register" \
  -H "Content-Type: application/json" \
  -d '{
    "username": "user123",
    "email": "user@example.com",
    "password": "pass123",
    "api_key": "PREMI-12345-67890-ABCDE-FGHIJ-KLMNO"
  }'
```

### Войти в аккаунт:
```bash
curl -X POST "http://127.0.0.1:8000/auth/login" \
  -H "Content-Type: application/json" \
  -d '{
    "username": "user123",
    "password": "pass123"
  }'
```

### Проверить свой аккаунт:
```bash
curl -X GET "http://127.0.0.1:8000/auth/me" \
  -H "X-API-Key: PREMI-12345-67890-ABCDE-FGHIJ-KLMNO"
```
